#¿como usar un commit desde la terminal, y que es un push y pull en git?
##User: cpmp usaar un commit

GitHub Copilot: Para usar un commit en Git, sigue estos pasos básicos:

1. **Agregar cambios al área de preparación (staging area)**: Usa `git add <archivo>` para agregar archivos específicos o `git add .` para agregar todos los cambios.

2. **Crear el commit**: Ejecuta `git commit -m "Mensaje descriptivo del commit"`. El mensaje debe ser claro y conciso sobre los cambios realizados.

3. **Verificar**: Usa `git log` para ver el historial de commits.

Ejemplo:
```
git add .
git commit -m "Agrega funcionalidad de login"
```

## Push y Pull en Git

### Push
El comando `push` envía tus commits locales a un repositorio remoto.

Comando básico:
```
git push origin main
```
- `origin`: Nombre del repositorio remoto (por defecto).
- `main`: Rama a la que enviar los cambios.

### Pull
El comando `pull` descarga cambios del repositorio remoto y los fusiona con tu rama local.

Comando básico:
```
git pull origin main
```
- Descarga y fusiona automáticamente los cambios remotos.
