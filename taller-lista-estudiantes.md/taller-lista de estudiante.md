## 2. 🚀 Creación del proyecto

Usen Vite (recomendado por su rapidez):

```bash
cd 05-05
npm create vite@latest
```

Completen los datos cuando se les solicite:

* **Nombre del proyecto:** `lista-estudiantes`
* **Framework:** React
* **Lenguaje:** JavaScript

Luego ejecuten:

```bash
cd lista-estudiantes
npm install
npm run dev
```
