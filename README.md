# Workspace# 🚀 My Workspace

Bienvenido a mi workspace personal. Este repositorio centraliza diversos proyectos, herramientas y configuraciones que utilizo en mi flujo de trabajo diario.

---

## 📂 Estructura del Proyecto

La organización del workspace se divide por categorías para facilitar la navegación:

| Directorio | Descripción | Estado |
| :--- | :--- | :--- |
| `/apps` | Aplicaciones completas y funcionales. | `En progreso` |
| `/packages` | Librerías y utilidades compartidas. | `Estable` |
| `/experiments` | Pruebas de concepto y borradores rápidos. | `Archivo` |
| `/docs` | Documentación técnica y notas personales. | `Continuo` |

---

## 🛠️ Tecnologías Principales

Este workspace utiliza un stack moderno enfocado en la productividad:

* **Lenguajes:** TypeScript, Python, Rust.
* **Frameworks:** Next.js, FastAPI.
* **Herramientas:** Docker, Terraform, PNPM (Workspaces).
